# StreamGate 1.5.2

[English](#english) | [简体中文](#简体中文)

## English

StreamGate 1.5.2 provides camera and video-source ingest, browser distribution, recording/replay, device management, monitoring, and integration APIs for Windows and Linux.

### Editions and Packages

- Standard: available as Windows installer/ZIP and Linux x86_64/aarch64 tar.gz packages; up to 5 channels can be used without requesting a license.
- Professional: available as a separate Windows installer/ZIP and Linux x86_64/aarch64 tar.gz package; Professional features require a trial or commercial license.

### Highlights

- RTSP, ONVIF, GB28181, SRT backhaul, NVR, encoder, and existing-stream ingest
- Browser playback through HTTP-FLV, WS-FLV, HLS, and WebRTC
- Live preview, video wall, recording plans, indexed recording search, and replay controls
- OpenAPI, WebHook, MQTT, monitoring, diagnostics, and offline licensing
- Professional vendor-SDK integration, transcoding/hardware codec paths, advanced GB28181, SRT workflows, and command center
- Improved reconnect ownership, replay control contracts, logging integration, installer behavior, and release diagnostics

Windows installers are unsigned. Windows may display a publisher warning during installation; verify the SHA256 checksum before running the installer.

## 简体中文

StreamGate 1.5.2 面向 Windows 和 Linux 提供摄像头与视频源接入、浏览器分发、录像回放、设备管理、运行监控和开放接口。

### 版本与安装包

- 标准版：提供 Windows 安装包/ZIP，以及 Linux x86_64/aarch64 tar.gz；5 路内无需申请授权即可使用。
- 专业版：提供独立的 Windows 安装包/ZIP，以及 Linux x86_64/aarch64 tar.gz；专业版能力需导入试用授权或商业授权。

### 主要更新

- 支持 RTSP、ONVIF、GB28181、SRT 回传、NVR、编码器和现有视频流接入
- 支持 HTTP-FLV、WS-FLV、HLS、WebRTC 浏览器播放
- 支持实时预览、值守大屏、录像计划、录像索引检索和回放控制
- 支持 OpenAPI、WebHook、MQTT、运行监控、日志诊断和离线授权
- 专业版支持厂商 SDK、软件转码与硬件编解码、高级 GB28181、SRT 和综合指挥中心
- 完善输入重连职责、远端/本地回放控制、日志集成、安装流程和发布诊断

Windows 安装包当前未进行代码签名，系统安装时可能显示发布者提示。运行前请先核对 SHA256 校验值。
