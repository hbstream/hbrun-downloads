## 中文说明

StreamCore SDK 1.3.1 提供面向 Demo 和评估集成的公开 SDK 包，覆盖 Windows、Linux、macOS、iOS、Android 和 .NET 运行环境。

### 下载内容

- Windows x86 / x64 ZIP 包
- Linux x86_64 / aarch64 tar.gz 包
- macOS / iOS XCFramework 封装包
- Android AAR 包
- .NET NuGet 包

### 主要能力

- RTSP / RTMP / HTTP(S) 拉流播放
- 摄像头、麦克风、屏幕和本地媒体采集
- RTMP / RTSP / SRT 推流发布
- GB28181 上级平台注册与媒体发送
- H.264 / H.265 解码、低延迟参数、状态回调和日志诊断

Demo 授权仅用于 Demo 工程验证，生产或商业分发请申请正式授权。

## English

StreamCore SDK 1.3.1 provides public SDK packages for demo evaluation and integration testing across Windows, Linux, macOS, iOS, Android, and .NET environments.

### Downloads

- Windows x86 / x64 ZIP packages
- Linux x86_64 / aarch64 tar.gz packages
- macOS / iOS XCFramework packages
- Android AAR package
- .NET NuGet package

### Highlights

- RTSP / RTMP / HTTP(S) playback
- Camera, microphone, screen, and local-media capture
- RTMP / RTSP / SRT publishing
- GB28181 upper-platform registration and media sending
- H.264 / H.265 decoding, low-latency options, status callbacks, and diagnostics
