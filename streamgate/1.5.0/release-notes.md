## 中文说明

StreamGate 1.5.0 面向摄像头接入、视频分发、录像回放和业务系统对接场景，提供 Windows ZIP 包和 Linux x86_64 / aarch64 tar.gz 包。

### 标准版

- 默认 5 路内免费使用
- 支持 RTSP / ONVIF / GB28181 / SRT 回传 / 现有视频流接入
- 支持 Web Console、本地通道管理、实时预览、录像计划、录像检索回放和值守大屏
- 支持 HTTP-FLV / WS-FLV / HLS / WebRTC 浏览器播放

### 专业版

- 未导入授权时按 5 路内免费标准能力运行
- 导入试用授权或商业授权后启用专业版能力
- 支持厂商 SDK 适配、SRT 回传、高级 GB28181、转码、硬件编解码、综合指挥中心和开放接口

### Windows 下载说明

当前 Release 提供 Windows ZIP 包。Windows installer 会在匹配的安装包发布后加入下载清单。

## English

StreamGate 1.5.0 is designed for camera ingest, video distribution, recording/replay, and business-system integration. This release provides Windows ZIP packages and Linux x86_64 / aarch64 tar.gz packages.

### Standard Edition

- Free for up to 5 channels by default
- Supports RTSP / ONVIF / GB28181 / SRT backhaul / existing stream ingest
- Supports Web Console, local channel management, live preview, recording plans, recording search/replay, and video wall
- Supports HTTP-FLV / WS-FLV / HLS / WebRTC browser playback

### Professional Edition

- Runs with the free 5-channel Standard baseline before a license is imported
- Enables Professional capabilities after importing a trial or commercial license
- Supports vendor-SDK adaptation, SRT backhaul, advanced GB28181, transcoding, hardware codec paths, command center, and integration APIs

### Windows Downloads

This release provides Windows ZIP packages. Windows installer packages will be added after a matching installer build is published.
