# StreamCore SDK 1.3.3

[English](#english) | [简体中文](#简体中文)

## English

StreamCore SDK 1.3.3 provides native and managed integration packages for Windows, Linux, Android, iOS, macOS, and .NET.

### Packages

- Windows x64 and x86 ZIP packages
- Linux x86_64 and aarch64 tar.gz packages
- Android AAR
- iOS arm64 framework archive
- macOS x86_64 framework archive
- .NET NuGet package

### Highlights

- RTSP, RTMP, and HTTP(S) playback with H.264/H.265 decoding
- Camera, microphone, screen, and local-media capture
- RTMP and RTSP publishing, plus licensed SRT publishing
- GB28181 registration to an upstream platform and media delivery
- ONVIF discovery, runtime status callbacks, configurable logging, and low-latency playback profiles
- Hardware decoding on supported platforms, with Professional direct-surface presentation paths

The packages carry demo-bound licensing for evaluation in the official demo applications. Production integration requires a matching commercial license.

## 简体中文

StreamCore SDK 1.3.3 提供 Windows、Linux、Android、iOS、macOS 和 .NET 集成包。

### 发布包

- Windows x64 与 x86 ZIP 包
- Linux x86_64 与 aarch64 tar.gz 包
- Android AAR
- iOS arm64 framework 压缩包
- macOS x86_64 framework 压缩包
- .NET NuGet 包

### 主要更新

- 支持 RTSP、RTMP、HTTP(S) 拉流播放及 H.264/H.265 解码
- 支持摄像头、麦克风、屏幕和本地媒体采集
- 支持 RTMP、RTSP 推流，以及授权后的 SRT 推流
- 支持向上级 GB28181 平台注册并发送媒体
- 支持 ONVIF 搜索、运行状态回调、日志配置和低延迟播放参数
- 支持平台硬件解码，专业版提供硬件解码直接呈现路径

公开包内置与官方演示程序绑定的 Demo 授权，用于功能评估。正式集成和商用交付需要申请对应商业授权。
